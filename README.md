# OpenClaw Dashboard

Dashboard de projetos e tarefas com drag and drop.

## Acesse online
https://jrcaporal-cloud.github.io/openclaw-dashboard

## Funcionalidades
- ✅ Drag and drop de tarefas entre colunas
- ✅ Reordenar colunas
- ✅ Adicionar novas tarefas
- ✅ Design moderno dark mode
- ✅ Totalmente responsivo
- ✅ Funciona offline

## Como usar
1. Acesse o link acima
2. Arraste tarefas entre as colunas
3. Clique em "Nova Tarefa" para adicionar
4. Use Ctrl+N para atalho rápido
